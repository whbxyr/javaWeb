# javaWeb
使用java开发接口
## 目录结构
```
  src                           —— java资源文件
  WebContent                    —— Web资源文件
    assets                      —— 前端静态资源，比如js、img等
    view                        —— jsp视图 模板
    WEB_INF                     —— javaWeb安全目录
      lib                       —— 所需要的jar包库
      springmvc-servlet.xml     —— springmvc的配置文件
      web.xml                   —— javaWeb配置文件
  .gitignore                    —— git忽略配置
  Learn.iml                     —— idea配置文件，为了方便，一同push上来了
  README.md                     —— 项目说明文件
```
